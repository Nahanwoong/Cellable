## Requirements

- Swift 4.2
- iOS 10.0+

## Installation

Cellable is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'Cellable', :tag => '1.1.0', :git => 'https://github.com/nibdevn/Cellable'
```
